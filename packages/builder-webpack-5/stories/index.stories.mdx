# Webpack 5
Для корректной сборки кита при использовании webpack 5 и выше необходимо воспользоваться билдером который расширит конфигурацию вашей сборки.

## Установка
```
npm i @wb-tech/builder-webpack-5 -D
```

## Использование
Импорт пакета.
```js
const extendWebpackConfig = require('@wb-tech/builder-webpack-5');
```

Конфигурация до расширения:
```js
// webpack.config.js
module.exports = {
    entry: {
        main: './src/index.js'
    },
    output: {
        path    : './dist',
        filename: '[name].bundle.js'
    },
    ...
};
```

Конфигурация после расширения:
```js
// webpack.config.js
const extendWebpackConfig = require('@wb-tech/builder-webpack-5');

module.exports = extendWebpackConfig({
    entry: {
        main: './src/index.js'
    },
    output: {
        path    : './dist',
        filename: '[name].bundle.js'
    },
    ...
}, {
    // Передать в опции MiniCssExtractPlugin если он используется в проекте
    useMiniCssExtractPlugin: MiniCssExtractPlugin
});
```

## Опции
- `disableTSRule` - Параметр `disableTSRule` используется для отключения правил обработки `ts`.
- `disablePCSSRule` - Параметр `disablePCSSRule` используется для отключения правил обработки `pcss`.
- `disableInclude` - Параметр `disableInclude` используется для отключения директорий обработки билдером `@wb-tech/ui-react`.
- `useMiniCssExtractPlugin` - Параметр `useMiniCssExtractPlugin` используется для передачи `MiniCssExtractPlugin` если в вашей сборке он используется.

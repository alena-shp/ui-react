<Heading>Design Tokens</Heading>
<Description>Предоставляет набор дизайн токенов в виде CSS переменных. Все предоставляемые переменные ограничены областью использования компонента или хука.</Description>
<Description>Содержит параметры: цветовая палитра, типографика, скругления, тени.</Description>

<Heading>Использование</Heading>
<Description>Импорт хука в код.</Description>

```tsx
import { useDesignTokens } from '@wb-tech/ui-react';
```

<Description>Хук возвращает объект с атрибутами для использования.</Description>

```tsx
import React from 'react';
import { render } from 'react-dom';

import { useDesignTokens } from '@wb-tech/ui-react';

const App = () => {
    const attrs = useDesignTokens();

    return (
        <div {...attrs}>
            <YourAppCode />
        </div>
    );
}

render(<App />, document.getElementById('root'));
```

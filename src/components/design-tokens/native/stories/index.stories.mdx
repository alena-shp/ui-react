<Heading>Design Tokens</Heading>
<Description>Предоставляет набор дизайн токенов в виде CSS переменных. Все предоставляемые переменные ограничены областью использования указанного элемента.</Description>
<Description>Содержит параметры: цветовая палитра, типографика, скругления, тени.</Description>

<Heading>Использование</Heading>

```tsx
import { useDesignTokensNative } from '@wb-tech/ui-react';
```

<Description>Устанавливает на указанный елемент класс с переменными дизайн системы.</Description>

```tsx
import { useDesignTokensNative } from '@wb-tech/ui-react';

useDesignTokensNative(document.getElementById('root'))
// или
useDesignTokensNative(document.body)
```
